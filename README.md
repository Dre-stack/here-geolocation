### To run project locally:

- Clone Repo
- Run `npm config set @here:registry https://repo.platform.here.com/artifactory/api/npm/maps-api-for-javascript/`
- Run `npm install`
- Run `npm start`

### For tests:

- Run `npm test`

* To show test coverage

- Run npm `test -- --coverage`
